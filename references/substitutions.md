# Exercise Substitution Rules

Use this when the user needs an exercise replacement or travel/no-equipment option.

## Core Rule

Replace the **movement pattern and training intent**, not merely the named exercise.

Output should include:

- Original exercise and likely training intent.
- Readiness status.
- Replacement exercise.
- Sets/reps/load/intensity.
- Pain rule.
- What to try next session.

## Pain-Based Substitutions

### Knee Pain

Avoid assuming all squats are bad. Reduce range, load, speed, or instability first.

Options:

- Squat pattern: box squat, sit-to-stand, goblet squat to box, wall squat partial range.
- Lunge pattern: reverse lunge partial range, supported split squat, step-up low box.
- Quad work: Spanish squat isometric if known and pain-free, terminal knee extension with band.
- Cardio: incline walk only if knee tolerates, flat walk, cycling, elliptical.

Avoid deep knee flexion, jump landings, fast direction change, or heavy lunges when pain is 3+/10.

### Low Back Pain

Reduce spinal loading and hinge complexity.

Options:

- Hinge: hip hinge drill, glute bridge, hip thrust, cable/band pull-through, supported RDL light.
- Pull: chest-supported row, one-arm row supported, band row.
- Core: dead bug, bird dog, side plank, suitcase carry light.

Avoid heavy RDL/deadlift, loaded flexion/rotation, or fatigue-driven form breakdown during symptoms.

### Shoulder Pain

Keep pressing in pain-free range and avoid forcing overhead work.

Options:

- Press: floor press neutral grip, incline press low angle, push-up incline, landmine-style press if available.
- Pull/support: chest-supported row, band row, face pull, rear delt fly light.
- Raise: lateral raise in scapular plane light, range-limited.

Avoid overhead press, deep dips, upright rows, or painful end range when symptoms appear.

### Wrist / Elbow Pain

Change grip and loading angle.

Options:

- Neutral-grip DB press/row.
- Push-up handles or fists if comfortable.
- Hammer curl instead of supinated curl.
- Bands/cables with neutral wrist.

Avoid high-volume gripping or loaded wrist extension if pain rises.

### Foot / Ankle / Calf Pain

Reduce impact and incline.

Options:

- Flat walking, cycling, elliptical, swimming.
- Calf isometric or light calf raise only if pain-free.
- Balance near support.

Avoid sprinting, jumping, steep incline, or fast direction changes until symptoms settle.

## Equipment Substitutions

- Barbell squat → goblet squat, box squat, split squat, step-up.
- Deadlift → DB RDL, hip thrust, glute bridge, band pull-through.
- Bench press → DB floor press, push-up incline, band press.
- Pull-up → band pulldown, inverted row, one-arm DB row, assisted pull-up.
- Cable row → band row, DB row, chest-supported row.
- Machine leg press → goblet squat, step-up, split squat, wall sit.

## Travel / No Equipment

Use lower-risk patterns:

- Sit-to-stand or squat to chair.
- Incline push-up.
- Hip hinge drill or single-leg RDL reach.
- Glute bridge.
- Towel row only if anchor is safe; otherwise skip.
- Dead bug, side plank, bird dog.
- Single-leg stance near support, heel-to-toe walk, suitcase carry if luggage is safe.
- Walking intervals, stairs only if knee/foot tolerates.

## Return-to-Original Rule

Return to the original exercise only when:

- Replacement is pain-free at 0–2/10.
- Next-day symptoms are not worse.
- Movement quality is stable.
- Load and range are reintroduced one variable at a time.
