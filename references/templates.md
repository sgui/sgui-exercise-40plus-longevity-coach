# Output Templates

Use these when the user needs a structured, reusable answer.

```md

<one-sentence job>

## Weekly Skeleton
| Day | Focus | Dose | Notes |
|---|---|---:|---|

## Strength Sessions

- Main 1:
- Main 2:
- Accessory:
- Core/balance:
- Stop rule:

### Session B
...

## Cardio / Zone 2
- Target:
- RPE/talk test:
- HR caveat:

## Progression Rule
<exact variable to progress first>

## Log
- Sleep/fatigue:
- Pain before/during/after:
- RPE/RIR:
- Notes:
```

## Today Adjustment Template

```md
## Readiness: <Green/Yellow/Orange/Red>
## Today’s Job
<one sentence>

## Workout
1. Warm-up — <time>
2. Main work — <exercise, sets/reps/time, load, rest, RPE/RIR>
3. Mobility/balance — <dose>
4. Cooldown / optional — <dose>

## Stop / Modify Rule
<pain and fatigue rule>

## Log Next
<what user should report>
```

## Substitution Template

```md
## Readiness: <Green/Yellow/Orange/Red>

Original: <exercise>
Intent: <pattern / muscle / training job>
Replacement: <exercise>
Dose: <sets/reps/load/RPE/rest>
Why: <short reason>
Pain rule: <0–2 continue, 3–4 modify, 5+ stop>
Next test: <how to decide next session>
```
